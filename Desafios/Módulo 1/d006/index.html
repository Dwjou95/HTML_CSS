Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Define um hiperlink
  • Define uma abreviação
  • Define um endereço
  • Define uma área dentro de um mapa de imagem
  • Define um texto em negrito(não semântica)
  • Define um texto em negrito(tem valor semântico)
  • Define uma base URL para todos os links da página
  • Define a direção do texto apresentado
  • Define uma citação longa
  • Define o corpo da página
